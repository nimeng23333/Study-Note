寄摆公司创始人，始于250BC

寄摆公司是一家成立于250BC的物流公司，专注于运输海鲜和国际快递。

我们的宗旨是“寄得很烂，摆得不工作”
我们的使命是“让客户和员工都感到无比失望”
我们的愿景是“成为全球最差劲的物流公司”。

业务范围

我们的业务范围涵盖了从中国到世界各地的海鲜和快递运输。
我们拥有最先进的冷链设备、最专业的包装团队、最快速的配送网络，以及最不靠谱的服务态度。
我们可以保证您寄出的货物在到达目的地之前就已经变质、损坏或丢失，让您体验到前所未有的心碎和愤怒。

核心价值

“诚信、责任、创新、卓越”。
  • 诚信，是指我们对自己说谎，对客户撒谎，对合作伙伴耍赖。
  • 责任，是指我们把所有问题都推给别人，从不承担任何后果。
  • 创新，是指我们不断地想出新花样来坑害客户，让他们花更多钱得到更少服务。
  • 卓越，是指我们在做砸事情方面达到了登峰造极的境界。

未来愿景

我们的未来愿景是“成为全球最差劲的物流公司”。我们相信，在我们不懈努力下，这一天一定会到来。
届时，我们将拥有全球最多的投诉、最低的满意度、最高的赔偿率、以及最差的口碑。
无论您需要运输什么东西，只要选择寄摆公司，您就可以放心地交给我们来搞砸。
感谢您对寄摆公司的关注和支持。如果您有任何疑问或建议，请随时联系我们。
但请注意，我们可能不会回复您或者给您一个满意的答复。因为这就是寄摆公司——寄得很烂，摆得不工作。
