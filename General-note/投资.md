# 1.投资前需要清楚的事
择时、择物

无非就是五个问题：1.买什么；2.什么时候买；3.买多少；4.什么时候卖；5卖多少
但这个问题涉及到多个方面的问题
上到政策、经济、流通性，下到个人情绪、个人资金总量多方面的影响。
而投资之所以难就是我们普通人没有时间和渠道去获取足够多的信息并进行筛选，然后做出选择。

# 2.寻找对手
交易涉及到买卖双方，对手盘。
那么一般对手有哪些
1.国家队（特点资金量大，行动有惯性，窗口期长，动向较为清晰）以安全为主
2.外部资金，资金量大，以盈利为主
3.公募基金，资金量大，更偏向龙头，由于大多组成是散户，因此情绪很影响
4.私募基金
5.游资，投机，妖股
6.量化，按照数据统计进行，以小票为主（因为根据统计，资金收益率会随着市值上涨而呈现边际递减，因此量化会选择收益率高的时期）
7.散户
8.结构性金融，包括雪球等产品

雪球：可以理解为一种场外开盘。假设有一个股票当前100元，未来走势不清楚，服务方与投资人约定：6个月后，只要6个月内没有超过80-103这个区间范围，那么6个月到期会兑现12%的收益，也就是112元。一旦超过103，按照时间结算收益，比如3个月就到了103，那么会兑现6%的收益。一旦低于80，那么服务方终止并返还投资人80元。也就是说风险由投资人承担。
雪球的盈利模式在于服务方可以在6个月内进行网格交易，通过做T来赚取波动收益。
会有一种情况当股票一直下跌，但却没有跌穿，到59天的时候仍然在81块钱，那么此时，服务方就会进行砸盘。这就是结构性金融产品的一种影响方式。

了解到对手的行为模式以后，低风险爱好者可以观察国家队很长的窗口期，确定它的趋势和惯性，从而进行交易。

# 3.调节指数的工具和调节情绪的工具。
在A股里，银行属于国有资产，且市值占比非常大（我估计20%）而且交易量很小。这意味着，假设比如工商银行涨1%只需要不大的资金量，而指数会上涨0.2%。而且对国家而言这个是左手倒右手，不是需要付出真金白银的。因此银行是调节指数的重要工具。

券商则是调节情绪的工具。
券商的盈利逻辑非常简单：资金量越大，指数越高，券商的效益就越好。是最简单的盈利模型。因此稍微有意识的投资者会认为券商如果都没涨上去，那么说明市场还不够热。现阶段国家通过压制住券商，来让股民情绪不要暴动。同时它也是一种国家工具。比如当美股出现某些重大挫折，比如跌20%，那么为了让国内股民不要恐慌，可以通过提升券商的价格，达到缓解情绪的目的（因为股民可能认为，外面虽然在跌，但国内还在涨，那么恐慌就影响不了国内股民）

而如何判断目前券商明明已经在历史价格高点了，却还是认为处在低估值区域呢？那就涉及到它的价值问题。价格不等于价值。
根据财报可以发现，比如公司盈利已经提升50%了，但2025年一年却只涨了7%，那么就可以大胆的判断它的价值高于它的实际价格。并不能单纯从以前的价格来推断未来的走势。

# 4.为什么我们需要这轮牛市
2024年6月降息+房价，导致当前市场没有商品能支撑起保险行业大约6%的成本，那么急需第三条路，也就是股市。

为什么降息和房价会影响保险呢？因为保险公司提供也是一种投资方式，给客户承诺3%的收益率的同时，保险经理的提成可以到10%左右，综合下来，客户每买100元的投资产品，那么保险公司的实际成本在106元左右，那么保险公司的投资收益就必须高于106。而保险公司也有很大的资金用于投资楼市。也就是说降息和房价下跌导致保险没有够本的收入了。

冷知识：目前没有任何一家保险公司破产。

保险破产会引发一系列的经济雪崩。

因此我们需要这轮牛市

# 5.什么是反内卷
反内卷本质是供需关系的改革。
现在产能严重过剩导致供远大于求。
国家号召反内卷
而煤炭、钢铁等国资，对国家号召响应更及时。而类似锂电池，光伏等私企更多的行业，想的更多的是熬死别人，所以对反内卷只是口头相应。
这就导致了化工行业内，不同内容的股价表现不同。煤炭上涨而锂电池下跌等。

# 6.科技
一句话，科技的确定性很强，内核很稳而扰动很多。
比如现在的科技债、私募贷都是加息敏感，对融资成本比较大。比如甲骨文评级调整为负面，与3000亿资本开支和大量融资相关。估值过高，资本开支很大，利润出问题就会出问题。

# 8.黄金
伦敦交易所等交易制度都是黄金准备金制度，也就是说可能会发生黄金实物挤兑情况。
现在其他各国都希望让黄金回流。实物黄金的回流能增加黄金上的话语权。
上海交易所正在进行黄金回收的动作。
目前的下跌或许是市场行为，也可能是利益相关者希望能缓解挤兑情况而做空。
但长期来看黄金的增值逻辑并未发生改变。
