---
title: 正则表达式匹配
---
10题

输入：
`s="mississippi"`
`p="mis*is*p*."`
无模式：mi
有模式：s*、p*、.

匹配无模式：
`[a-z]+(?=([a-z.]\*|$))`
第一个[a-z]+代表匹配任意多个字母字符
括号是捕获符号
?=意思是跟着后面的内容，但不包括后面的内容
a-z. 加上一个`*`或者结尾

有很多坑
比如
```
s="aaa"
p="a*a"
```

```
s="ab"
p=".*c"
```

```
s="aab"
p="c*a*b"
```

最坑如下：`b*`和`c*`虽然都能被匹配为0
但两变的a无论是从左到右还是从右到左，都会因为`a*`而发生错误
```
s="aaa"
p="ab*a*c*a"
```
