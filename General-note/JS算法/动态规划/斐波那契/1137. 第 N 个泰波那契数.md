---
title: 1137. 第 N 个泰波那契数
---
泰波那契序列 Tn 定义如下： 

`T0 = 0`, `T1 = 1`, `T2 = 1`, 且在 `n >= 0` 的条件下 `Tn+3 = Tn + Tn+1 + Tn+2`

给你整数 `n`，请返回第 n 个泰波那契数 Tn 的值。

**示例 1：**

**输入：** n = 4
**输出：** 4
**解释：**
```
T_3 = 0 + 1 + 1 = 2
T_4 = 1 + 1 + 2 = 4
```

**示例 2：**

**输入：** n = 25
**输出：** 1389537

```js
var tribonacci = function(n) {
    let t0 = 0, t1 = 1, t2 = 1;
    for(let i = 3; i< n+1; i++){
        if(i % 3 === 0){
            t0 += t1 + t2
        }
        if(i % 3 === 1){
            t1 += t0 + t2
        }
        if(i % 3 === 2){
            t2 += t0 + t1
        }
    }
    let res;
    if(n % 3 === 0){
        res = t0
    }
    if(n % 3 === 1){
        res = t1
    }
    if(n % 3 === 2){
        res = t2
    }
    return res
};
```
