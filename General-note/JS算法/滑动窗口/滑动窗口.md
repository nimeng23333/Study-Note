恰好型滑动窗口可转变为2个滑动窗口
恰好k个，转为至少k个 以及至少k+1个
k+1 个 减去k个就是恰好k个的答案

至少k个：
```js
function countSub(nums, k){
	let left = 0, max = Math.max(...nums),count = 0
	for(let i = 0; i< nums.length; i++){
		if(nums[i] === max) count++
		while(count >= k){
			if(nums[left] === max) count--
			left++
		}
		res += left
	}
	return res
}
```
