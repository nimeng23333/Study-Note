---
title: 3.冒泡排序
---
从头到尾依次比较相邻两个元素的大小
```js
	function bubbleSort(arr){
		let length = arr.length
		while(length > 0){
			for(let i = 0; i< length-1;i++){
				if(arr[i] > arr[i+1]){
					let temp = arr[i]
					arr[i] = arr[i+1];
					arr[i+1] = temp
				}
			}
			length--
		}
		return arr
	}
```

冒泡排序的效率为O(N^2)
