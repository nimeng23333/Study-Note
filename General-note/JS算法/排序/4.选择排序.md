---
title: 4.选择排序
---
选择排序将交换次数从O(N^2)减少到O(N)但比较次数依然是O(N^2)，效率优于冒泡排序
思路：
选定第一个索引的位置，和后面元素依次比较
如果后面队员小于第一个位置，就交换，使用同样方法把剩下的元素逐个比较就可以。
选择排序第一轮会选出最小值
```js
function selectSort(arr){
	let round = 0
	while(round < arr.length){
		for(let i = round+1; i< arr.length; i++){
			if(arr[round] > arr[i]){
				let temp = arr[round]
				arr[round] = arr[i]
				arr[i] = temp
			}			
		}
		round++
	}
	return arr
}
```
