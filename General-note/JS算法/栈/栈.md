# 栈在十进制转二进制的应用

比如100转2进制
```
100 % 2 = 0
50 % 2 = 0
25 % 2 = 1
12 % 2 = 0
6 % 2 = 0
3 % 2 = 1
1 % 2 = 1
```

1100100
结果是后进先出，用栈计算

```js
function DectoBin(num){
	const stack = new Stack() //自己封装一个栈类
	while(num > 0){
		stack.push(num % 2)
		num = Math.floor(num / 2)
	}
	let str = ''
	while(!stack.isEmpty){
		str += stack.pop()
	}
	return str
}
```
